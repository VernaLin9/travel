# 名古屋夏日之旅 9天8夜（防暑調整版）

單一檔案網頁（`index.html`），無任何外部相依，部署於 GitHub Pages：
**<https://vernalin9.github.io/travel/>**

家庭成員：2 大 2 小（6 歲・7 歲，一台推車）。

## 功能
- **三種閱讀模式**：卡片（看重點）、時間軸（上午／正午／傍晚防暑節奏）、表格（總覽）
- **🔀 換行程助手**：選「今天第幾天」＋「狀況（太熱／下雨／小孩沒電／多出時間）」→ 立刻給替代方案、路線、下一步接法
- **📖 景點攻略**：每個景點的玩法、門票優惠、期間限定活動（2026 夏）、必買、美食、親子推車資訊
- **防暑對策**：時段安排原則、裝備清單、推車防暑、中暑警訊
- **室內備案雷達**＋**美食清單**
- **響應式**：手機單欄、電腦多欄自動切換
- **列印版**：按右上「🖨️ 列印版」或 Ctrl+P，輸出精簡 A4 版面（行程總表＋裝備＋備案＋限定活動速查）

## 開啟 GitHub Pages
Repo 的 **Settings → Pages** → Source 選 **Deploy from a branch** → Branch 選 `main` / `(root)` → Save，等 1–2 分鐘即上線。

## 修改行程
所有行程資料都集中在 `index.html` 內 `<script>` 開頭的 `DAYS`、`BACKUPS`、`FOODS` 三個陣列，直接改文字即可，不需要懂程式。
